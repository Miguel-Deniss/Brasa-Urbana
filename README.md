# Brasa Urbana - Site Restaurante Premium

Site profissional desenvolvido para o restaurante fictício **Brasa Urbana**, criado como case de portfólio para demonstrar a criação de sites para restaurantes e pequenos negócios gastronômicos.

## Tecnologias

- **React** + **TypeScript**
- **Tailwind CSS v4**
- **Framer Motion**
- **Lucide React**
- **Vite**

## Funcionalidades

- Design premium e responsivo (mobile-first)
- Navegação suave com header fixo
- Cardápio com filtros por categoria
- Galeria com lightbox
- Carrossel de depoimentos
- Botão WhatsApp flutuante com mensagem pré-preenchida
- Animações de scroll (fade-in)
- SEO básico (Open Graph, meta tags)
- Lightbox de imagens

## Como executar

```bash
npm install
npm run dev
```

## Build de produção

```bash
npm run build
npm run preview
```

## Estrutura

```
src/
├── components/   # Componentes do site
├── data/         # Dados (cardápio, depoimentos, galeria)
├── App.tsx       # Componente principal
├── main.tsx      # Entry point
└── index.css     # Estilos globais e tema Tailwind
```
